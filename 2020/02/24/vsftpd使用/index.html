vsftpd的使用

参考文档

安装

键入命令：sudo apt-get install vsfptd

为了能够正常使用ftp_TLS，需要在/etc/vsftpd.conf中添加：

# 开启 ssl 并指定使用的协议
ssl_enable=YES
ssl_tlsv1=YES
ssl_sslv2=NO
ssl_sslv3=NO
# 指定证书和 key 文件
rsa_cert_file=/*自己的目录*/*证书*.crt
rsa_private_key_file=/*自己的目录*/*证书*.key
# 安全选项
allow_anon_ssl=NO
force_local_data_ssl=YES
force_local_logins_ssl=YES
require_ssl_reuse=NO
ssl_ciphers=HIGH
# 指定主动模式时使用的端口范围
pasv_min_port=*自己设置端口*
pasv_max_port=*自己设置端口*

重启vsftpd服务即可。

设置账户

参考文档

需要添加账户，并指定主目录：sudo useradd -d /*自己设置目录*/ -s /usr/sbin/nologin *用户名*。之后设置密码：passwd *用户名*

之后通过chown *用户名*:*用户名* *目录*设置文件夹权限。

下面完成两方面的设置：

  • 只允许特定用户登陆
  • 只允许读写主目录
userlist_enable=YES
userlist_deny=NO
userlist_file=/etc/vsftpd/user_list

chroot_local_user=YES
chroot_list_enable=YES
chroot_list_file=/etc/vsftpd/chroot_list
allow_writeable_chroot=YES

其中，chroot_list中的用户可以访问主目录外的其他目录。

乱码问题

如果出现乱码问题，可以通过将文件系统字符集设置为utf-8解决。在/etc/vsftpd.conf中设置：utf8_filesystem=YES。在filezila中[站点]中的[字符集]强制使用utf-8编码即可。
